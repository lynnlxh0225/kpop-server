# K-pop 随舞管理器 v2 部署指南

> v2 数据模型变化：用户 + 好友 + 车主，**不再有"小组"概念**。每个用户有自己的邀请码，被邀请的人成为好友。歌曲的发起人是"车主"，只有车主能改歌曲/排练/路演信息，组员只能改自己的出席。

### A.3 服务器上：清空数据库 + 重启

SSH 登录服务器：

```bash
ssh lynn@47.93.9.199
```

执行：

```bash
cd ~/kpop-server

# 1. 停止后端
pm2 stop kpop-server

# 2. 备份旧数据库（万一以后想看一眼）
mkdir -p ~/backups
mv data/data.db ~/backups/data_v1_$(date +%Y%m%d_%H%M%S).db 2>/dev/null

# 3. 清空 data 目录（v2 会自动重建）
rm -f data/data.db data/data.db-wal data/data.db-shm

# 4. 启动 v2（会自动 init 新表结构）
pm2 start kpop-server
pm2 logs kpop-server --lines 20
```

应该看到 `✅ 数据库初始化完毕` 和 `🚀 K-pop server v2 running on http://127.0.0.1:3000`。

## 路径 B：全新部署

### B.1-B.5 步骤同上一份"后端部署指南"的第 1-6 步

唯一区别：把 `kpop-server` 换成 `kpop-server-v2`：

```bash
# 本机
cp -R ".../outputs/kpop-server-v2" ~/Desktop/
scp -r ~/Desktop/kpop-server-v2 lynn@47.93.9.199:~/kpop-server

# 服务器
cd ~/kpop-server
npm install
mkdir -p logs data
cp .env.example .env
openssl rand -base64 48     # 复制输出
nano .env                    # 改 JWT_SECRET，保存
pm2 start ecosystem.config.cjs
pm2 save
pm2 startup                  # 跟着提示执行返回的命令

# nginx 配置（如果还没配置过）
sudo tee /etc/nginx/sites-available/kpop > /dev/null << 'EOF'
server {
    listen 8080;
    server_name _;
    root /home/lynn/kpop-server/public;
    index index.html;
    location / { try_files $uri $uri/ /index.html; }
    location /api/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
EOF
sudo ln -sf /etc/nginx/sites-available/kpop /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx
```

## 故障排查

### 浏览器显示老版本（没有登录页）
- 强制刷新（`Cmd + Shift + R`）
- 检查 `cat ~/kpop-server/public/index.html | head -5` 确认是新版（应能看到登录注册相关 HTML）

### 注册按钮点了没反应
- 浏览器 `F12` → Console 看错误
- 服务器 `pm2 logs kpop-server` 看后端日志

### "邀请码无效"
- 邀请码区分大小写要小心 — 后端会自动转大写，但你复制时可能多了空格
- 检查对方邀请码是否被重置过（在「⚙️ 设置」可以查到）

### 想清空所有数据重来
```bash
pm2 stop kpop-server
rm -f ~/kpop-server/data/data.db*
pm2 start kpop-server
```

## 与 v1 关键区别速查

| 维度 | v1 | v2 |
|------|-----|-----|
| 组织模式 | 小组共享所有数据 | 用户独立 + 好友关系 |
| 邀请码 | 小组的，全员共用 | 每人一个 |
| 谁能编辑歌曲 | 任何成员 | 只有车主 |
| 谁能编辑排练/路演 | 任何成员 | 只有车主 |
| 谁能改出席状态 | 任何成员可改任何人 | 只能改自己（车主创建时可批量预填） |
| 加成员 | 从小组成员里选 | 从自己好友里选 |
| 退出歌曲 | 删除即移除 | 标记 `left`，保留历史只读访问 |
